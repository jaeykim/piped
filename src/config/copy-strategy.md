# Copy Strategy (카피 전략)

소구점 생성, 카피 선택, 후킹 전략의 핵심 규칙입니다.
"메타 퍼포먼스 광고 A to Z"에서 추출한 원칙 기반.

## 핵심 원칙

1. **메타는 충동구매** — 검색 의도가 아니라 충동을 만들어야 한다
2. **콘텐츠 = 성과의 80%** — 운영은 20%, 콘텐츠가 구리면 뭘 곱해도 0
3. **고객 언어 사용** — 공급자 언어(기능/스펙) 금지
4. **0.5초 안에 "이거 내 얘기네"** — 타겟의 정확한 상황을 묘사
5. **메인→서브 인과관계** — 문제→해결 또는 결과→근거

## 공급자 언어 vs 고객 언어

| 공급자 언어 ❌ | 고객 언어 ✅ |
|--------------|------------|
| 검색동선과 전환동선 최적화 | 신점이 무서워서 도전 못하셨던 분들! |
| 마케팅 자동화 플랫폼 | 아직도 수동으로 광고 만드세요? |
| AI 기반 실시간 데이터 분석 | 광고비 절반으로 매출 2배 |
| 퍼포먼스 마케팅 솔루션 | 3분 만에 광고 완성 |

## 소구점(Appeal Point) 유형

### hook (후킹)
0.5초 안에 스크롤을 멈추게 하는 문구.
타겟의 정확한 상황을 묘사해야 한다.

패턴:
- `[상황에 처한 타겟]이 나 몰래 쓰던 [제품]`
- `아직도 [타겟의 고통스러운 현재]하고 계세요?`
- `[놀라운 결과]인 줄 알았는데 [제품]이었습니다`
- `[구체적 숫자]만에 [결과]` (예: "61세도 100만원씩 벌 수 있는")

### benefit (혜택)
고객이 얻는 결과를 고객의 언어로. 반드시 구체적 숫자 포함.
- NOT 기능 → 결과 ("하루 833원으로 해결", "매출 2배, 시간은 절반")

### proof (증거)
신뢰를 만드는 구체적 증거.
- 실제 숫자, 후기 스타일, 인증/수상 언급

## 성공 요소 분리 (고도화 패턴)

잘 된 콘텐츠에서 성공 요소를 분리하고, 왜 먹혔는지 해석:
- **높은 연령** → "나도 당연히 되겠지" (난이도↓ 인식)
- **계좌 사진** → "진짜 돈이 들어온다" (증거)
- **구체적 금액** → "100만원" (기대치 설정)
- **"99% 모르는"** → "경쟁 없다" (희소성)
- **"누구나"** → "진입장벽 없다" (접근성)

## 카피 구조

### 메인 카피 (헤드라인)
- MAX 3-5단어
- 타겟의 상황을 묘사 (기능 설명 X)
- 숫자 포함 시 전환율 상승

### 서브 카피 (서브헤드라인)
- 메인을 보완하는 인과관계
- 메인이 문제 → 서브가 해결책
- 메인이 결과 → 서브가 근거

### CTA
- 행동 유도 문구 (할인율, 혜택, 긴급성)
- 예: "지금 바로 50% 할인 받기", "무료로 시작하기"

## 타겟 정의 규칙

인구통계학이 아니라 가치관 + 상황 + 행동으로 정의:
- BAD: "20대 여자 직장인"
- GOOD: "미래가 궁금해서 신점을 보고 싶지만 무서운 분위기가 꺼려져서 보지 못했던 사람들"

타겟 정의가 뾰족할수록 → 메시지가 뾰족 → 머신러닝이 정확한 사람을 찾음
